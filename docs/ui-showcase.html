UI Showcase - v1.0

BertyType

Local, offline voice dictation for Windows. Captures microphone audio, transcribes via VibeVoice STT, optionally refines with Gemma 4, and injects text system-wide - no cloud, no latency.

100% Offline Push-to-Talk LLM Refinement System-wide Injection VibeVoice STT Gemma 4 via Ollama
